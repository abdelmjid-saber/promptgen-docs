# Localization
By default is set in English but you can change it on some of the available languages.
This option you will find in ِ ِ **Admin** > **Languages**.

## Add New Language

Since some of the languages are still not available you can add manually new language. There are several steps you need to do for achieving that using Translation plugin available with the admin account.w
This option you will find in ِ ِ **Admin** > **Languages**.
