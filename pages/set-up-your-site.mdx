# Set up your site
## Change Logo & Favicon

* Login as admin
* Go to Settings -> Logo & And Favicon
* Upload yourl logo & favicon

## Change the project logo and default images
Go to your project and login as admin. 
In the settings page, you will be able to change the project logo and the default images for the menu items and the cover image in the restaurant menu.

## SEO Configuration
* GO to Settings -> SEO Configuration

## Change the content of the front page
* Go to Settings -> SEO Configuration

## Resolve the issue of images not showing up
To resolve the issue of images not showing up, you can try the following steps:
* Go to **https://example.com/symlink.php** 
* Once the page loads, close it
* Check if the images now appear successfully.
